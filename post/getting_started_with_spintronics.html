Python
Python is a versatile programming language that allows you to work quickly and integrate systems more effectively. It's an essential tool for automating tasks, managing, and visualizing data. Here's how you can get started with Python:

Step 1: Installing Python

  • Visit the official Python website at python.org and download the latest version for Windows.
  • Open the downloaded installer. Ensure to check the box that says Add Python 3.x to PATH before clicking "Install Now".
  • Open Command Prompt and type python --version to verify that Python is installed correctly.

Step 2: Installing itcpr python package

  • Create a folder named Research, where all your research files will be stored.
  • Open 'Windows Command Prompt' (CMD) and type cd your/path/to/the/research/folder, and press Enter.
  • Type pip install itcpr --upgrade and press Enter.
  • Minimize this CMD window.

Git
Git is a distributed version control system that allows you to track changes in your codebase and collaborate with others. Here's how you can get started with Git:

Step 1: Installing Git

  • Visit Git's official website at git-scm.com and download the latest version for Windows.
  • Run the installer and follow the installation prompts. Ensure to select the default options unless you have a specific reason not to.
  • Open Command Prompt and type git --version to verify that Git is installed correctly.
  • Type git config --global user.name "Your Name" and git config --global user.email "youremail@gmail.com" to configure your Git username and email.
Complete Setup


Step 1: Copying Folders from Github

  • Don't proceed further if you're not added to our organization on Github.
  • Open the minimized CMD window from the Python installation step.
  • Type python and press Enter.
  • Type import spintronics as sp and press Enter. If you don't see any errors, you have successfully installed the itcpr package.
  • Type sp.start_learning() and press Enter.
  • It'll ask for your name ID. Copy the name ID from email you received from ITCPR.

Step 2: Understanding Folders

  • In your Research folder, you'll see four folders: learning, mumax3, python-codes, and yournameID. You'll see a file named jupyter.bat, which opens Jupyter Notebook.
  • The learning folder contains tasks for your training period that are mendatory for you to complete to start working in projects.
  • The mumax3 folder contains sample mumax3 codes, which will be useful for learning our simulation codes.
  • The python-codes folder contains sample python codes, which will be useful for your research journey in our institution.
  • The yournameID folder will contain your personal files for ressearch. You can create subfolders and store your research files in this folder. Inside this folder, there's a file named z_git_push.bat, which will push your files to Github.
