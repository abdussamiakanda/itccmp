Introduction
This ultimate guide is designed to help you securely connect to our organization's remote server from your personal computer. It's crafted for users with varying levels of technical expertise, ensuring everyone can successfully set up and use ZeroTier, WinSCP, Putty, and Remote Desktop.
Preparing Your Computer
  • Ensure your Windows is up to date to avoid compatibility issues.
  • Have your server credentials ready: server IP, username, and password. These will be provided by ITCPR.
  • Ensure you have administrative rights on your PC for software installations.
Part 1: Establishing a Secure Connection with ZeroTier
ZeroTier creates a virtual network over the internet, simulating the experience of being directly connected to the remote server's local network.

Step 1: Downloading ZeroTier

  • Visit ZeroTier's download page.
  • Select the Windows version and download the installer.
  • Run the installer and follow the installation prompts.

Step 2: Joining the Network

  • Launch ZeroTier from the Start Menu or System Tray.
  • Navigate to the 'Networks' tab and click 'Join Network'.
  • Enter the network ID provided by ITCPR.
  • Let us know you have joined the network to receive network approval.

Step 3: Verifying Connectivity

  • Check the status in the ZeroTier application; a successful connection is indicated by "OK".
  • If the status does not show "OK", try restarting ZeroTier or checking with ITCPR for network approval.
Part 2: File Transfer with WinSCP
WinSCP allows for secure file transfer between your PC and the server.

Step 1: Installing WinSCP

  • Visit WinSCP's download page.
  • Run the installer and follow the installation prompts.

Step 2: Connecting to the Server

  • Open WinSCP and fill in the session details:
    • Host name: server.itcpr.org
    • User name: (Provided by ITCPR)
    • Password: (Provided by ITCPR)
    • Port number: 22
  • Click 'Login' to establish the connection.
  • If unable to connect, verify network connectivity and credentials.
Part 3: Command Line Access with PuTTY
PuTTY provides SSH access for executing commands on the server.

Step 1: Downloading and Installing PuTTY

  • Navigate to PuTTY's download page and download the MSI installer.
  • Install PuTTY by following the on-screen instructions.

Step 2: SSH into the Server

  • Open WinSCP and go to 'Options' >> 'Preferences' >> 'Applications'.
  • Check the option 'Remember session password and pass it to PuTTY (SSH)'.
  • Press 'Ctrl + P' or click on the 'Open session in PuTTY' icon on the top.
  • If the connection fails, check the server IP, port, and network status.
