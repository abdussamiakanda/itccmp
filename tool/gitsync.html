What is GitSync?

GitSync is a lightweight, system-tray application that silently keeps your Git repositories in sync. Built by ITCPR, it’s ideal for researchers and developers who work across multiple machines and want an always-synced local copy.

Download GitSync

You can download the latest release of GitSync for Windows below. The app is packaged as a single `.exe` file.

Download GitSync
How to Use
  1. Download and run the GitSync executable.
  2. It will automatically:
    • Add itself to system startup
    • Run in the background
    • Appear in your system tray
  3. Right-click the tray icon for:
    • Projects — Edit the Git URLs in projects.json
    • About — View this page
    • Quit — Exit the app
How to Add Repositories

Open the projects.json file from the tray menu and add your repositories like this: 

{
  "repos": [
    "https://github.com/your-username/project1.git",
    "https://github.com/your-username/project2.git"
  ]
}

GitSync will clone any missing repositories, and every minute, it will:

  • Pull from the remote if there are changes
  • Push local changes if you've modified files

Requirements
  • Python 3.6+ (installed on the system)
  • Git must be installed and available in your system PATH
  • Credentials should be cached (e.g., via Git Credential Manager or GitHub CLI)
